Tallify

Created using Semantic UI, NodeJS, Postgres

To reduce the likelihood of having the value of a $20 voucher being reduced to zero after a 'magical' date, I created a tracking system for users to tally the vouchers they have so that they can easily view the vouchers they have.

Mingle (Team project)

Created using Ruby on Rails, Semantic UI, Postgres

To make it easy for people to form groups and connect through LinkedIn, we created a networking web-app where event organisers can create events, and participants can then form groups using it. One of the motivations for this app is an actual scenario where we witnessed how someone had difficulty in remembering the LinkedIn id/email to share, and this app is our attempt to solve this problem.

Bubble Pop Game

A nostalgic web-based pattern game modelled after the 90s game machines. The interface is deliberately kept clean and simple so that the focus is on remembering the patterns.
